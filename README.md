# Incident Management Website

Klar MkDocs/Material-version af et komplet videnssite om incident management.

## Åbn i Visual Studio Code

1. Åbn mappen i VS Code  
2. Installer Python og MkDocs efter behov  
3. Installer afhængigheder:

```bash
pip install mkdocs mkdocs-material mkdocs-blog-plugin
```

4. Start lokalt:

```bash
mkdocs serve
```

5. Byg sitet:

```bash
mkdocs build
```

## Struktur

- `mkdocs.yml` – navigation og tema
- `docs/` – alt indhold i Markdown

## Formål

Dette projekt er skrevet til at kunne bruges direkte i repoet `IncidentManagementWebSite` eller kopieres ind som en ny version.
